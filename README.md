# Tuchus
Tuchus is a client-friendly Trailhead Override for [Statamic's](http://statamic.com/) backend.

## Changes
* removed the wilderness-themed elements,
* stuck to a monochrome pallette, and 
* changed all of the fonts to sans-serif.
